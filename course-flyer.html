Class Goal:

This class it to provide kids with 101 knowledge about web designs. At the end of this class, they will build their own (and the first) public website hosted by ICEA, so everyone on the internet can click and see them.

Class Plannings:

Lesson #1: (Introduction into the internet - how websites work)

In this lesson we are going to learn some basic concepts of the web or Internet. We use the basic browser tool to inspect the webpage behind the scene.

Lesson #2 (Coding a web page):

In this lesson we will learn the basic structure of a web page and have some experiments on coding a web page.

Lesson #3 (Styling your webpage):

In this lesson, we will learn how to make a website look more beautiful.

Lesson #4 (Styling your web page - Continued):

In this lesson, we will continue to make the web page cooler and more playful.

Lesson #5 (Styling your web page with a library - Bootstrap):

In this lesson, we will learn how to reuse other peopleâ€™s work (a library) to make your web page look more professional.

Lesson #6 (Introduction to JavaScript):

In this lesson, we will learn a more powerful language to teach your web pages to be more intelligent.

Lesson #7 (Debugging with JavaScript)

In this lesson, we will build up some basic debugging skills to shape your coding with javascript.

=========================================== 2 week gap ===========================================

Lesson #8 (Final Project Showcase):

In this class, we will give everyone a chance to show off their final project. We will expect every student to have 5 minutes to present their websites (final project), explain what web technologies they used, and share interesting features with other students.
